# NVimWordDefinition
This is a plugin for Neovim that allows you to define words.

## Installation
In Neovim if use vim-plug then add this to your init.vim
```vim
Plug 'elvisclaros/NVimWordDefinition'
```
Then run `:PlugInstall` in Neovim.

## Usage

`:ShowDefinition`.
